# 💰 Tip Calculator

A simple Tip Calculator built using HTML, CSS, and JavaScript.  
This application helps users calculate the tip amount, total bill, and the amount each person needs to pay when splitting the bill.

## 🚀 Features

- Calculate tip based on bill amount and tip percentage
- Split the bill among multiple people
- Instant calculation using JavaScript
- Simple and clean user interface
- Responsive design

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript

## 📚 Key Concepts Used

- DOM Manipulation
- Event Handling
- User Input Handling
- Basic JavaScript Calculations
